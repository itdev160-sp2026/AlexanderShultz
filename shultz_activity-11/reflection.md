# Activity 11 Reflection: React Tic-Tac-Toe

## Key Learnings

React breaks down code into reusable components, allowing use to reduce reused code. It also simplifies state management, automatically updates DOM. Parent components pass data to their own children, allowing a one-way flow for data. However, arrays must be copied using slice(), allowing for data in arrays to be more secure than others.

## Comparison

Vanilla JS requires manual DOM, alongside not being able to reuse code. Alongside the fact that JS uses event listeners, something like time travel will be hard to impliment. React automatically updates DOM, reusing code through it's props. It's a lot more clear to use, and can be powerful once learned.
